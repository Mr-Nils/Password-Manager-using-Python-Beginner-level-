# 🔐 CLI Password Manager (Python)

A simple terminal-based password manager built in Python that allows users to securely store and manage their credentials.

## 🚀 Features

* Save new passwords
* Retrieve passwords for a given site
* Update existing passwords
* View all stored entries
* Generate strong random passwords
* Simple and interactive CLI interface

## 🛠️ Tech Used

* Python 3
* File Handling (Text)
* `secrets` module for password generation

## ▶️ How to Run

```bash
git clone https://github.com/your-username/password-manager.git
cd password-manager
python app.py
```

## 📌 Example Usage

* Enter a website name → get saved password
* Add new credentials
* Generate secure passwords instantly

## ⚠️ Note

This is a basic educational project. Passwords may not be encrypted and should not be used for storing real sensitive data.
